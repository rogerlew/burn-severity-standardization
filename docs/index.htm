WEPPcloud Soil Burn Severity Map Search

Final Soil Burn Severity maps have been acquired from burnseverity.cr.usgs.gov and automatically standardized using this package (https://github.com/rogerlew/burn-severity-standardization/).

The maps have NOT been extensively validated and users are cautioned to be wary of potential errors from the standardization process.

The baer-db does not have a way to hardlink to a fire and does not have mtbs-ids. The search is matching ignition dates. Please double-check the baer-db result is the fire you're looking for.

For questions, comments, or feedback on these datasets please contact: rogerlew@uidaho.edu
